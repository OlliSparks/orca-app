# PMS - Produktmanagement Sonderzubehör

Rolle für das Produktmanagement Sonderzubehör.

## Steckbrief

| Eigenschaft | Wert |
|-------------|------|
| **Kürzel** | PMS |
| **Englisch** | Product Management Special Accessories |
| **Typ** | Process |
| **Launch Phase** | LP1 |
| **Abbildung** | Über "ABO" (Abonnement) |

## Berechtigungen

| Berechtigung | Beschreibung |
|--------------|--------------|
| **View** | Lesezugriff |
| **Approve** | Genehmigungsrechte |

## Abbildung über ABO

Diese Rolle wird nicht als feste Gruppenmitgliedschaft abgebildet, sondern über das **Abonnement-System (ABO)**. Benutzer mit besonderem Interesse an bestimmten Assets/Prozessen können sich diese abonnieren.

## Kontext

Das Produktmanagement Sonderzubehör ist verantwortlich für spezielle Zubehörteile und deren Werkzeuge.

## Hauptaufgaben

1. **Produktüberwachung** - Monitoring von Sonderzubehör-Assets
2. **Prozessbeteiligung** - Beteiligung an relevanten Prozessen
3. **Genehmigung** - Freigabe im Sonderzubehör-Bereich

## Workflow-Beteiligung

Der PMS wird bei relevanten Prozessen automatisch über das ABO-System benachrichtigt und kann bei Bedarf Genehmigungen erteilen.
