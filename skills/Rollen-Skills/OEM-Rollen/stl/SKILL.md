# STL - Steuerliche Abwicklung (Local)

Rolle für die steuerliche Abwicklung mit Zugriff auf den eigenen Buchungskreis beim OEM.

## Steckbrief

| Eigenschaft | Wert |
|-------------|------|
| **Kürzel** | STL |
| **Englisch** | Tax Settlement (Local) |
| **Typ** | Rolle |
| **Launch Phase** | LP2 |
| **Organisation** | OEM |
| **Scope** | Nur eigener Buchungskreis |

## Berechtigungen

| Berechtigung | Beschreibung |
|--------------|--------------|
| **View** | Lesezugriff auf relevante Daten |
| **Approve** | Genehmigungsrechte |

## Status

**Rolle unklar - Abklärung ausstehend**

## Hauptaufgaben

1. **Lokale steuerliche Prüfung** - Prüfung steuerrelevanter Asset-Daten im eigenen Buchungskreis
2. **Lokale Genehmigung** - Freigabe aus steuerlicher Sicht für lokalen Bereich

## Abgrenzung zu STC

| Aspekt | STL (Local) | STC (Commodity) |
|--------|-------------|-----------------|
| Buchungskreis-Scope | Nur eigener | Alle |
| Verantwortungsbereich | Lokal | Konzernweit |
| Launch Phase | LP2 | LP1 |
