# REV - Revision

Rolle für allgemeine interne Interessenten (Wirtschaftsprüfer, Revision) beim OEM.

## Steckbrief

| Eigenschaft | Wert |
|-------------|------|
| **Kürzel** | REV |
| **Englisch** | Revision / Audit |
| **Typ** | Gruppe |
| **Launch Phase** | LP1 |
| **Organisation** | OEM |
| **Authentifizierung** | BMW LDAP |

## Berechtigungen

| Berechtigung | Beschreibung |
|--------------|--------------|
| **View** | Nur Lesezugriff |

## Zugriff

- **Dashboard** - Zugriff auf Dashboards
- **Reports** - Zugriff auf Reports

## Zielgruppe

- Wirtschaftsprüfer (WP)
- Interne Revision
- Weitere interne Interessenten

## Hauptaufgaben

1. **Audit** - Prüfung der Asset-Daten
2. **Compliance-Kontrolle** - Überprüfung der Einhaltung von Vorschriften
3. **Reporting** - Auswertung von Reports

## Authentifizierung

Die REV-Gruppe wird über BMW LDAP bereitgestellt - keine manuelle Benutzerverwaltung in Orca erforderlich.

## Einschränkungen

- Kein Schreibzugriff
- Keine Genehmigungsrechte
- Nur Dashboard und Reports

## Ähnliche Rollen

| Rolle | Unterschied |
|-------|-------------|
| RM (Risikomanagement) | Ähnlicher Lesezugriff, Fokus auf Risiko |

## Abonnement-Option

Benutzer mit besonderem Interesse können auch über ein Abonnement Zugriff erhalten (View-Berechtigung).
